
## Exportacao PDF com jsPDF + Compartilhamento

### Resumo

Substituir o `window.print()` por geracao real de PDF usando jsPDF + jspdf-autotable. Incluir coluna `unit_price` (ja existe no banco mas nao e usada no frontend). Adicionar compartilhamento via Telegram alem dos ja existentes (WhatsApp e email).

---

### 1. Instalar dependencias

- `jspdf` - geracao de PDF client-side
- `jspdf-autotable` - plugin para tabelas automaticas

---

### 2. Atualizar modelo de dados para incluir `unit_price`

**`src/hooks/usePurchaseOrders.ts`**:
- Adicionar `unit_price: number` na interface `PurchaseOrderItem`
- Incluir `unit_price` nos payloads de create/update

**`src/components/purchases/PurchaseOrderForm.tsx`**:
- Adicionar campo de preco unitario em cada item row
- Atualizar `ItemRow` interface com `unit_price`

**`src/components/purchases/PurchaseOrderDetail.tsx`**:
- Adicionar colunas "Preco Unit." e "Total" na tabela de itens
- Exibir total geral da OC (soma dos `quantity * unit_price`)

---

### 3. Criar funcao de geracao de PDF

**Criar `src/lib/generatePurchaseOrderPdf.ts`**:

Funcao `generatePurchaseOrderPdf(order, projectName)` que:

1. **Cabecalho**: Logo ObraBoa (inline SVG convertido para imagem), nome do projeto, data, numero da OC
2. **Dados do fornecedor**: Nome e contato em bloco de texto
3. **Tabela de itens** (via autoTable):
   - Colunas: #, Descricao, Qtd, Unid., Preco Unit. (R$), Total (R$)
   - Linha de total ao final
   - Estilo com header em azul navy (#1B3A5C), linhas zebradas
4. **Observacoes**: Bloco de texto abaixo da tabela (se existir)
5. **Rodape**: "Gerado por ObraBoa" + data/hora + numero da pagina
6. Retorna `jsPDF` instance para `.save()` ou `.output('blob')`

Layout A4 portrait. Fonte padrao (Helvetica). Formatacao monetaria BR (R$ 1.234,56).

---

### 4. Atualizar PurchaseOrderDetail

**`src/components/purchases/PurchaseOrderDetail.tsx`**:
- Substituir `handlePrint` por chamada a `generatePurchaseOrderPdf`
- Adicionar opcao "Telegram" no dropdown de compartilhamento
- `handleShare` recebe `'whatsapp' | 'telegram' | 'email'`
- Telegram usa `https://t.me/share/url?url=&text={encoded}`
- O texto de compartilhamento inclui totais (se `unit_price > 0`)

### Detalhes Tecnicos

- **Logo no PDF**: Converter o SVG para base64 data URL e usar `doc.addImage()`. Como o SVG contem `<text>` que o jsPDF nao renderiza bem, converter para PNG base64 hardcoded ou usar apenas texto "ObraBoa" estilizado como fallback.
- **autoTable**: Importar como `import autoTable from 'jspdf-autotable'`. Configurar `startY`, `styles`, `headStyles` com cor #1B3A5C, `alternateRowStyles`.
- **Formatacao monetaria**: `new Intl.NumberFormat('pt-BR', { style: 'currency', currency: 'BRL' }).format(value)`.
- **Compartilhamento com totais**: O `buildSummary` incluira `ðŸ’° Total: R$ X.XXX,XX` quando houver precos.
