# ObraBoa — Contexto do Projeto

## O que é
Plataforma mobile-first de gestão de obras para o mercado brasileiro.
Público: house flippers, pequenos escritórios de arquitetura, clientes finais.

## Stack
- React + TypeScript
- Tailwind CSS
- Supabase (backend/auth)
- React Router DOM

## Design System
- Cor primária: #0D3259 (navy)
- Cor de destaque: #F59E0B (amber)
- Fonte: Nunito
- Mobile-first: breakpoint principal 390px (iPhone 14)
- Estilo de referência: FamilyWall — cards limpos, bottom navigation, 
  empty states amigáveis, FAB (floating action button)

## Princípios de UX
- Máxima simplicidade — usuário sem experiência técnica
- Curva de aprendizado mínima, sem necessidade de treinamento
- "Essência Apple": simples, bem elaborado, bonito sem ser complexo
- Navegação em 3 níveis: Dashboard → Lista → Detalhe
